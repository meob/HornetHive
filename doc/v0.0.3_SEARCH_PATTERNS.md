# 🛰️ HORNET HIVE | v0.0.3 Implementation Plan: Search Patterns

## 1. Objective
Enable autonomous Search and Rescue (SAR) patterns for drones. This feature allows an operator to dispatch a drone to a point and then initiate an automated search (Expanding Square / Spiral) without manual waypoint management.

## 2. User Interface (UI) Updates
- **File**: `public/index.html`
- **Component**: Mission Control Panel (sidebar).
- **Change**: 
    - Reduce the width of the "MOVE TO TARGET" button.
    - Add a small square button (e.g., 40x40px) next to it.
    - **Icon**: `🌀` (Spiral Symbol) or a custom CSS/SVG spiral.
    - **Behavior**: Clicking the button sends a `SEARCH` command via MQTT if the drone is currently active (Hovering, Loitering, or Transit).

## 3. Communication Protocol (MQTT)
- **Topic**: `hive/swarm/target`
- **Command**: `SEARCH`
- **Payload Example**:
```json
{
  "target_id": "ALPHA",
  "command": "SEARCH",
  "mission": "SAR",
  "x": 1200, 
  "y": -500, 
  "z": 80,
  "pattern": "SPIRAL_SQUARE",
  "step": 100
}
```
*Note: `x`, `y`, `z` will be the coordinates of the current drone's target.*

## 4. Drone Navigation Logic (`drone_mock.py`)
### 4.1 New Internal State
A `SearchController` class or a set of variables within `drone_mock.py` to manage navigation:
- `search_center`: (x, y) coordinates where the search began.
- `search_step`: Distance between "lanes" (default 100m, or altitude-dependent).
- `current_leg`: Index of the current segment of the spiral.
- `leg_length_limit`: Maximum distance for the current leg.
- `leg_distance_traveled`: Tracking progress on the current segment.

### 4.2 Expanding Square Geometry
1. **Leg 1**: North (Length: `search_step`)
2. **Leg 2**: East (Length: `search_step`)
3. **Leg 3**: South (Length: `search_step * 2`)
4. **Leg 4**: West (Length: `search_step * 2`)
5. **Leg 5**: North (Length: `search_step * 3`)
... and so on.

### 4.3 Real-world Integration
- **Rotary**: The drone will "snap" to corners or perform coordinated turns.
- **Fixed-Wing**: The drone will attempt to intercept the next leg using its `stall_speed` and `orbit_radius` to ensure aerodynamic stability (no 90-degree instant turns).
